#+title: Problem with proxies
#+roam_key: Algorithms to live by

As we have seen, the use of proxy metrics - taste as a proxy for
nutrition, number of cases solved as a proxy for investigator
diligence - can also lead to overfitting. In these cases, we'll need
to cross-validate the primary performance measure we're using against
other possible measures.
